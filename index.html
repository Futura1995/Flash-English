基本的な使い方
  1. Google AI StudioでAPIキーを取得（無料）
  2. 画面上部のAPIキー欄に貼り付けて「保存」
  3. 「今日の気分」でデッキ（学習セット）を選択
  4. 日本語フレーズを見て英語を考える
  5. テキスト入力または録音で英訳を入力
  6. 「Geminiに添削してもらう」でフィードバックを確認
連続学習日数について

毎日学習を続けると🔥連続日数がカウントされます。
日付の切り替えは午前3時（深夜0〜2時台の学習は前日扱い）。
フレーズのナビゲーションまたは添削を行うと学習記録されます。

CSVインポートについて

「フレーズ管理」タブからCSVファイルをインポートできます。
1列目：日本語、2列目：英語ヒント、3列目：カテゴリ（省略可）
インポートしたグループは「今日の気分」のデッキとして選択できます。

APIキーの取得方法

1. aistudio.google.com にアクセス
2. 左メニュー「Get API key」→「Create API key」
3. 発行された「AIza...」のキーをコピー
4. このアプリ上部の入力欄に貼り付けて保存

音声認識について

録音機能はChromeブラウザで最もよく動作します。Safari・Firefoxでは動作しない場合があります。その場合はテキスト入力をご利用ください。
